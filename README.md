# Minimal Replication for CasMez 
Repositiory contains: 
`casmez_replication.csv` : Minimal Dataset from final .dta 
`Replication_Fig4.do` : Stata code to reproduce figure 4 in paper - CasMez Border RD 

To replicate: 
- Download files into the same folder.
- In Stata, open the `.do` file.
- Edit `global PATH` line at the top of your own folder.
- Run the code - final output should be a pdf of figure 4.
